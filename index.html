About I-Detector

The most flexible Application of creating image datasets

I-Detector is an application that provides a mean of creating image datasets that can be used in inteligent computer vision algorithms to solve different problems like image classification, detection or segmentation.i-detector is part of an ambitious project that aims to optimize/automate the maintenance of road/rails/buildings infrastructures.
